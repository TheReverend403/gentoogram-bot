Just a bot for the [@Gentoogram](https://t.me/Gentoogram) Telegram group. It filters chat with regex and stuff.

## Using

```
poetry install
cp settings.default.yml settings.yml
$EDITOR settings.yml
poetry run python -m gentoogram
```
